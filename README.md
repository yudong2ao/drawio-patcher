# DrawIQ 性能优化与自适应补丁 v2.1.0 (drawio-patcher2.0 优化版)

本补丁在官方原版 `draw.io` 桌面客户端的基础上，实现了**极速首屏启动、自适应深浅色过渡、以及 Windows 无边框自定义控件**，并提供安全的本地一键打包安装机制。

---

## ✨ 核心优化特性

### 1. ⚡ 启动性能压榨（首屏耗时缩短 60%+）
* **并发资源加载**：重构了 `bootstrap.js` 中原本长达 8 个 JS 文件的串行嵌套回调。采用 HTML5 的并行下载并严格顺序执行技术，消除磁盘 I/O 串行阻塞。
* **首屏体积减负**：将首屏冷启动时不需要的 12.2 MB 图形模版包（`stencils.min.js` 等 3 个大文件）移出首屏主队列，延迟到主界面渲染就绪 50ms 后在后台静默加载，**首屏文件编译体积减少 56%**。
* **按需动态加载**：将原版启动时预载的 Mermaid / ELK 绘图引擎剥离，仅在用户插入该图表时才在运行时动态离线唤起。

### 2. 🖥️ Windows 窗口无边框与防闪烁
* **纯净无边框（Frameless）**：主进程隐藏系统原生的标题栏（`frame: false`），视觉感官更加现代纯净。
* **无干扰显示时机**：主进程默认隐藏窗口，在渲染层 `ready-to-show` 首帧完全就绪后再延迟展示并恢复最大化，彻底杜绝了 Windows 冷启动瞬间黑屏/白屏的短暂闪烁。
* **自定义控制按钮**：在菜单栏右侧优雅地注入了最小化、最大化/还原、关闭 3 个悬浮控制按钮，点击与窗口状态同步。

### 3. 🎨 自适应深浅色过渡（用户配置优先）
* **双重防线检测**：Loading “加载中”画面和启动背景底色通过 `index.html` 同步自运行脚本（第一防线）和 `preload.js`（第二防线）进行双重大范围自适应检测。
* **用户主题优先**：优先读取 `.drawio-config` 中的 `darkMode` 配置，如果用户在软件内部 Extras 菜单中设置了暗黑或浅色，Loading 页面将精准呈现对应色调（`#202020` 或 `#f5f5f5`），不被操作系统偏好强制劫持。如果用户未配置，才自动跟随系统主题。

## 📦 补丁打包与安装说明

由于打包脚本 `drawio-patcher.js` 默认会直接对官方系统保护目录（如 `C:\Program Files`）下的原生 `app.asar` 进行备份和修改，因此运行前必须使用**管理员权限**：

1. **一键应用补丁**：
   * 在 Windows 搜索栏输入 `CMD` 或 `PowerShell`，右键点击并选择 **“以管理员身份运行”**。
   * 切换目录（`cd`）到本 `drawio-patcher2.0` 文件夹下，执行命令：
     ```bash
     node drawio-patcher.js
     ```
   * 脚本会自动在系统中定位官方安装的 `app.asar`，并在同目录下自动将其备份为 `app.asar.bak`，随后自动完成解包、补丁物理覆盖以及重新封包。
2. **如何还原官方原生版**：
   * 若未来需要还原官方原生版本，只需在 draw.io 安装的 `resources/` 目录下（通常为 `C:\Program Files\draw.io\resources\`）将 `app.asar.bak` 重命名回 `app.asar` 覆盖即可。
